# EstruturaDados-Projeto1-CRUDAlunos

Passo 1

Crie uma struct contendo as seguintes informações: identificador (ID), Nome do aluno e nota do aluno.

Passo 2

Crie uma lista encadeada que permita as seguintes operações:
Cadastro de um novo aluno com as informações dele.
Remoção de um aluno pelo identificador.
Mostrar aluno com determinado identificador.
Editar um aluno buscando ele pelo identificador.
Mostrar todos os alunos da lista.
Mostrar as seguintes estatísticas para a lista de alunos: Maior nota, menor nota e média das notas dos alunos.

Passo 3

Deverá ser criado um menu seguindo as opções acima e ao final de cada operação o menu será apresentado novamente. A aplicação deverá ser fechada quando o usuário escolher a opção 0. 

Para as funcionalidades de remoção (2), mostrar (3) e editar (4), caso seja informado o ID de um aluno que não exista na lista a operação não deverá seguir e deverá ser mostrado “Usuário não encontrado”.
